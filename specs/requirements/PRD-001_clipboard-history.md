## 1. 背景・目的

Macの標準クリップボードは1件しか保持できず、過去にコピーした内容を再利用するには再度コピーし直す必要がある。
本ツールは最大30件のクリップボード履歴を保持し、キーボードショートカット一発で呼び出せるようにすることで、作業効率を向上させる。
自分自身の利用に加え、他のMacユーザーへのDMG形式での配布も前提とする。

---

## 2. スコープ

### In Scope（対象）
- クリップボード履歴の自動取得・保存（テキスト・画像）
- 最大30件の履歴保持（起動中のみ、再起動でリセット）
- Commandキー2回押しによるポップアップ起動
- マウスカーソル付近へのポップアップ表示
- 履歴アイテム選択によるクリップボード内容の変更（貼り付け可能な状態にする）
- DMGファイルによる配布（署名・公証対応）

### Out of Scope（対象外）
- クリップボード履歴の永続化（再起動後も保持）
- テキスト・画像以外のデータ形式（ファイル、URLカード等）
- 履歴の検索・フィルタリング機能
- クラウド同期
- App Store配布
- Windows / Linux対応

---

## 3. ユーザーストーリー

```
As a Macユーザー
I want to 過去にコピーしたテキストや画像を素早く呼び出したい
So that 同じ内容を何度もコピーし直す手間を省き、作業効率を上げる
```

```
As a Macユーザー
I want to Commandキーを2回押すだけでポップアップを表示したい
So that マウス操作なしに直感的に履歴へアクセスできる
```

## 4. 機能要件

### FR-001: クリップボード監視
- **説明**: アプリ起動中、NSPasteboardの変更を常時監視してテキスト・画像をキャプチャする
- **優先度**: Must
- **受け入れ条件**:
  - [ ] テキストのコピーを自動で検知して履歴に追加される
  - [ ] 画像のコピーを自動で検知して履歴に追加される
  - [ ] 同一内容の連続コピーは重複して追加されない
  - [ ] アプリ自身がクリップボードを変更した場合は履歴に追加されない

### FR-002: 履歴管理（最大30件）
- **説明**: 履歴は最大30件を上限とし、超過した場合は最古のものを削除する
- **優先度**: Must
- **受け入れ条件**:
  - [ ] 履歴が30件を超えた場合、最も古いアイテムが自動削除される
  - [ ] 履歴はアプリ終了時にすべて破棄される（永続化しない）
  - [ ] 履歴は新しいものが上に表示される

### FR-003: グローバルショートカット（Commandキー2回押し）
- **説明**: システム全体で有効なグローバルホットキーとして、Commandキーを素早く2回押すとポップアップを起動する
- **優先度**: Must
- **受け入れ条件**:
  - [ ] どのアプリがフォーカスを持っていても反応する
  - [ ] 2回押しの判定間隔は0.3秒以内とする（定数で管理）
  - [ ] Accessibility権限がない場合は権限要求ダイアログを表示する

### FR-004: ポップアップUI（マウス付近に表示）
- **説明**: ポップアップウィンドウをマウスカーソルの付近に表示し、クリップボード履歴の一覧を見せる
- **優先度**: Must
- **受け入れ条件**:
  - [ ] ポップアップはマウスカーソルの近く（上または下）に表示される
  - [ ] 画面外にはみ出さないよう位置を自動調整する
  - [ ] テキストアイテムはプレビューとして先頭100文字を表示する
  - [ ] 画像アイテムはサムネイルとして表示する
  - [ ] Escキーまたはポップアップ外クリックで閉じる
  - [ ] ポップアップを閉じた後、元のアプリにフォーカスが戻る

### FR-005: 履歴アイテムの選択・貼り付け
- **説明**: ポップアップ内でアイテムを選択するとクリップボードに設定され、そのまま貼り付け可能な状態になる
- **優先度**: Must
- **受け入れ条件**:
  - [ ] アイテムをクリックするとクリップボードに設定される
  - [ ] 選択後、ポップアップが自動で閉じる
  - [ ] 選択後、元のアプリにフォーカスが戻る
  - [ ] キーボードの上下矢印キーで選択移動、Enterキーで決定できる

### FR-006: メニューバー常駐
- **説明**: アプリはDockに表示せず、メニューバーにアイコンとして常駐する
- **優先度**: Must
- **受け入れ条件**:
  - [ ] Dockにアイコンが表示されない
  - [ ] メニューバーアイコンをクリックするとメニューが表示される
  - [ ] メニューから「履歴をクリア」「終了」が実行できる

### FR-007: DMG配布パッケージ
- **説明**: 知り合いへ手渡しできるDMGファイルを生成できる（署名・公証なしの個人配布）
- **優先度**: Must
- **受け入れ条件**:
  - [ ] .appをDMGにパッケージングできる
  - [ ] 受け取り側が「右クリック → 開く」の手順で起動できることを確認済み
  - [ ] 配布時に初回起動の手順を説明するREADMEを同梱する

## 5. 非機能要件

| カテゴリ | 要件 | 計測指標 |
|---------|------|---------|
| パフォーマンス | クリップボード変更の検知から履歴追加まで < 200ms | 目視確認 |
| パフォーマンス | ポップアップ表示までの時間 < 100ms | 目視確認 |
| メモリ | 通常使用時のメモリ消費 < 50MB | Instruments |
| メモリ | 画像は元データ（Data）＋極小サムネイルのみ保持。NSImageをそのまま複数保持しない | コードレビュー |
| 安定性 | 24時間連続稼働でクラッシュなし | 手動テスト |
| 互換性 | macOS 13 Ventura 以降をサポート | 動作確認 |
| 権限 | Accessibility権限のみ要求（マイク・カメラ等は不要） | 権限設定確認 |

---

## 6. 制約・前提条件

- 対象OS: macOS 13 Ventura 以降
- 配布形式: DMG（知人への個人配布、署名・公証なし）
- 受け取り側は初回起動時に「右クリック → 開く」が必要
- Accessibility権限が必要（グローバルキーボード監視のため）
- 履歴データは揮発性（アプリ終了で消える）
- 開発言語: Swift / SwiftUI / AppKit

---

## 7. 用語定義

| 用語 | 定義 |
|-----|------|
| クリップボード履歴 | コピー操作でキャプチャしたテキスト・画像のリスト |
| ポップアップ | Commandキー2回押しで表示されるフローティングウィンドウ |
| グローバルホットキー | フォーカスに関わらずシステム全体で反応するキーボードショートカット |
| Gatekeeper | 未署名アプリの起動を警告するmacOSのセキュリティ機能。右クリック→開くで回避可能 |
