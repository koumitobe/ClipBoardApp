# ClipBoardApp — クリップボード履歴ツール

Macのクリップボード履歴を最大30件保持し、Commandキー2回押しで素早く呼び出せるメニューバー常駐アプリです。

---

## インストール手順

1. `ClipBoardApp.dmg` をダブルクリックしてマウント
2. 表示された `ClipBoardApp.app` を **アプリケーションフォルダ** にドラッグ
3. DMGをアンマウント（取り出し）

---

## 初回起動手順（重要）

このアプリは個人配布のため、macOSのGatekeeperにより通常の方法では起動できません。
**以下の手順で起動してください。**

1. アプリケーションフォルダの `ClipBoardApp.app` を **右クリック**
2. メニューから **「開く」** を選択
3. 「開発元を確認できません」というダイアログが表示される
4. **「開く」** ボタンをクリック

※ 2回目以降は通常通りダブルクリックで起動できます。

---

## 初回権限許可（重要）

起動後、キーボード監視のためにアクセシビリティ権限が必要です。

1. 起動直後に権限要求ダイアログが表示される
2. **「システム設定を開く」** をクリック
3. システム設定 → プライバシーとセキュリティ → アクセシビリティ
4. `ClipBoardApp` のトグルを **オン** にする
5. アプリを再起動する

---

## 使い方

| 操作 | 動作 |
|---|---|
| **Commandキーを素早く2回押し** | クリップボード履歴のポップアップを表示 |
| **↑ / ↓ キー** | 履歴アイテムを選択 |
| **Enterキー** | 選択したアイテムをペースト |
| **Escキー** | ポップアップを閉じる |
| ポップアップ外をクリック | ポップアップを閉じる |
| メニューバーアイコン → 履歴をクリア | 履歴を全件削除 |
| メニューバーアイコン → 終了 | アプリを終了 |

---

## 仕様

- 履歴保持件数: 最大30件（アプリ終了でリセット）
- 対応コンテンツ: テキスト・画像
- 動作環境: macOS 13 Ventura 以降
- Dock非表示（メニューバーのみ常駐）
